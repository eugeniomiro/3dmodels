# 3D Models repo

Este repositorio contiene todo lo relacionado con mi pasaje por la impresión 3D. 
En él podemos encontrar varios modelos rescatados de varios repositorios de modelos (Thingiverse, Cult3d, Printables, etc.) que imprimí o modifiqué,
además de los que yo creé para mis propósitos individuales.
Además, también se encuentran las historias de mis [impresoras](Printers/README.md), las que guardo para documentación propia y también por si a alguien le interesa consultarlas.
